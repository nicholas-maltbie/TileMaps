    Class Heap<K, V>

    Heap data structure that can store any given item for a specific value. Supports storing a generic set of elements by a given key value where the element with the smallest value is stored at the root of the tree. This is an implementation of the min heap structure via an array. The array can be specified size when starting up but will grow if more elements are added.
